# Structure Metadata and pass it to the wallet in Callback



****

Use [**Schema.org**](http://schema.org) specifications to structure metadata in form of **JSON-LD.**

eg.

```
var Metadata = {};
Metadata = {
 "type": "AudioObject",
 "name": "title",
 "creator": "artist",
 "image": "image" 
 }
 export var Metadata;
```
